# Dodge the Creeps

A simple 2D game where you dodge enemies. Good to get an overview of basic gdzig concepts.

## Build

1. Clone this repository:

```bash
git clone https://github.com/Kaiter-Plus/dodge-the-creeps-gdzig
```

2. Navigate to the project directory:

```bash
cd dodge-the-creeps-gdzig
```

3. Build the game:

```bash
zig build -Dgodot-version="<godot_version>" -Dgodot-path="<godot_executable_path>"
```

For example:

```bash
zig build -Dgodot-version="4.6" -Dgodot-path="D:\godot\godot.exe"
```
